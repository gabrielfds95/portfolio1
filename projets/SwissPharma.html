Le projet SwissPharma


Présentation du projet


Le projet Swiss Pharma est un projet de cours réalisé avec deux collègues et moi. Notre équipe s'est engagée à répondre aux exigences d'un client fictif, Swiss Pharma, opérant dans le secteur pharmaceutique. L'objectif était de développer un site web sécurisée et conviviale conçue pour simplifier la gestion des frais engagés par les visiteurs médicaux et assurer le suivi des remboursements. Nous avons utilisé la méthode Agile scrum et en combinant nos compétences en développement web et en conception d'interface utilisateur, nous avons créé une solution web "applis frais".



Technologies Utilisées

  • Le projet SwissPharma est développé en utilisant Langage de programmation : PHP, JavaScript
  • Tout le projet est relié à la base de donnée SQL avec PhpMyAdmin.
  • Utilisation des librairies font awesome et Bootstrap.


Gestion et Organisation

Nous avons utilisé la méthode agile scrum pour itérer rapidement et répondre efficacement aux besoins du client. Nous avons divisé le projet en sprint chacun durant 2 semaines. Dans chaque sprint on mettait à jour le planning (diagramme de gant), diagramme de classe, les maquettes, manuel d'utilisateur, documents de tests et évidemment le développement du site.

Voici comment se présente le site d'appli frais de SwissPharma :
