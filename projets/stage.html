Présentation de l'entreprise


Le projet Swiss Pharma est un projet scolaire réalisé par groupe de trois. Notre groupe s'est attelé à répondre aux besoins d'un client fictif de l'industrie pharmaceutique, Swiss Pharma. Notre objectif était de développer un site web dédié à la gestion efficace des fiches de frais des agents par le service comptable. En utilisant la méthode Agile et en combinant nos compétences en développement web et en conception d'interface utilisateur, nous avons créé une solution web "applis frais".



Technologies Utilisées

  • Langage de programmation : PHP, JavaScript
  • Base de données : MySQL, SQL
  • Librairies : font awesome, Bootstrap


Fonctionnalités

  • Page de connexion pour les utilisateurs avec différents niveaux d'accès.
  • Page de visualisation de la liste des fiches de frais avec options de filtre et de modification.
  • Page de détail d'une fiche de frais avec formulaire de modification.
  • Page d'ajout de nouvelles fiches de frais.
  • Page de profil pour la gestion des informations personnelles des utilisateurs.
  • Page d'aide avec FAQ organisée en accordéon.


Code du Projet

Dans ce projet, nous avons mis en œuvre une architecture MVC (Modèle-Vue-Contrôleur) pour garantir une structure organisée et modulaire de notre code. Notre stack technologique comprenait PHP pour le back-end, JavaScript pour la logique front-end, Bootstrap pour l'interface utilisateur, et MySQL pour la gestion de la base de données.


Documentation et Tests

Nous avons utilisé la méthode agile pour itérer rapidement et répondre efficacement aux besoins du client. Nous avons donc découpé le projet sous forme de sprint. Pour chaque sprint, nous avons fait une maquette, un planning, des tests fonctionnels. Nous avons également amélioré au fur et à mesure des sprints notre analyse, notre guide utilisateur et notre diagramme de classe.
